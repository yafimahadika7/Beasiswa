Sistem Informasi Pendaftaran Beasiswa berbasis web yang dibangun menggunakan

- Admin
- Reviewer
- Mahasiswa

**Teknologi yang Digunakan**
- PHP Native
- MySQL
- HTML, CSS, Bootstrap
- JavaScript
- Apache (XAMPP / Hosting)

**Struktur Database**
Struktur database digunakan untuk mengelola data pengguna, pendaftaran,
dan penilaian beasiswa.
